Mengenal Semantic ELEMENTS di HTML 5

  • Artikel

    • Ketika ingin membuat layout untuk menampilkan block artikel-artikel yang ada di website kita,tag article bisa kita gunakan dengan element header dan footer.karena tag article memiliki tipe block,artinay element ini memiliki lebar sesuai lebar browser jika tidak menenyukan lebarnya

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama.Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan untuk menjelasakn identitas penulis artikel.karena aside memiliki type/sifat block maka lebnarnya sesuai page jika kita tidak menentukan lebarnya
